Инструкция для **PHP + MySQL на ISPmanager** в этом репозитории **больше не поддерживается**:
код синхронизации переведён на **Supabase**.

Актуальные шаги: см. **deploy-supabase.txt** и **README.md**.

Если нужен снова свой PHP API — восстановите вручную endpoint и замените вызовы
в src/app/api/backend.ts на fetch к вашему серверу.
